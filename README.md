# netcapx

netcapx provides GUI-based network capturing tool using Python and Tkinter with Scapy for packet sniffing.

### Usage
1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
2. Run tha application:
   
   ```bash
   python netcapx.py
   ```
